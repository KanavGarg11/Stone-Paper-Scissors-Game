# 🎮 Stone Paper Scissors Game

A fun and interactive **Stone Paper Scissors** web game built using **HTML, CSS, and JavaScript**. This project allows players to compete against the computer with an animated selection process, score tracking, and a clean user interface.

---

## 🚀 Features

* 🎯 Play Stone, Paper, Scissors against the computer
* 🤖 Random computer move generation
* 🎞️ Animated computer "thinking" effect
* 🧮 Live score tracking for player and computer
* 🔄 Reset button to restart the game
* 🎨 Interactive UI with hover effects and winner highlighting
* 📱 Simple and responsive web layout

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the game
* **CSS3** – Styling and animations
* **JavaScript (Vanilla JS)** – Game logic and interactivity

## 🎮 How to Play

1. Click the **Play...** button.
2. The computer randomly selects a move.
3. Wait for the computer to lock its move.
4. Choose **Stone**, **Paper**, or **Scissors**.
5. The winner is displayed and scores are updated.
6. Click **Play Again** to continue.

---

## 🧠 Game Rules

* **Stone beats Scissors**
* **Scissors beats Paper**
* **Paper beats Stone**
* Same choices result in a tie
